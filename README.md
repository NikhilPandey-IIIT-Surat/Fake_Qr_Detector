# Fake QR Detector

A full-stack project using React, FastAPI, and OpenCV to detect fake QR codes.

## Features
- Upload QR image
- Webcam scanning
- Fake/Real detection
- Blur detection using OpenCV

## Tech Stack
- React + Vite + Tailwind
- FastAPI
- OpenCV + pyzbar

## How to Run

### Backend
cd backend
uvicorn main:app --reload

### Frontend
cd frontend
npm run dev
